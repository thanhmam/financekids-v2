Small rounded tag/label — age groups, lesson categories, "Bài 3/45" counters, filter pills.

```jsx
<Chip tone="green">6-8 tuổi</Chip>
<Chip tone="violet">13-16 tuổi</Chip>
<Chip tone="gold" solid>SALE</Chip>
```

Tones: `green`, `indigo`, `violet`, `gold`, `streak`, `neutral`. `solid` fills with the accent (white text); default is a soft tint.
