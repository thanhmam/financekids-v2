Rounded square holding an emoji/glyph icon — used on lesson cards, shop items, task rows, topic tiles.

```jsx
<IconTile bg="#FFF9E6" fontSize={22}>💰</IconTile>
<IconTile bg="var(--xu-gems-50)" color="var(--xu-gems)">◆</IconTile>
```

Tint `bg` (and `color` for glyph icons). `size` 40/46/52 are typical. Lesson cards pass the lesson's own `bgColor`.
